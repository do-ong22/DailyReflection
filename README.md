# 📋 일일 업무 회고 리포트 봇

Google 데이터(일정, 메일, Drive)를 자동으로 수집하여 Discord로 리포트를 전송합니다.

## 🚀 시작하기

### 1. Bot 초대하기

Discord 서버에 Bot을 초대합니다. (Bot을 만든 사람이 호스팅 중이라면 바로 사용 가능)

### 2. 처음 설정 (처음 1회)

```
/로그인
```

- `/로그인`만 입력하면 됩니다
- 브라우저가 열리면 Google로 로그인
- 새 노트북이 자동으로 생성됩니다

### 3. 리포트 생성

```
/시작
```

오늘의 일정, 메일, Drive 데이터를 수집하여 Discord에 리포트가 전송됩니다.

## 명령어 모음

| 명령어 | 설명 |
|--------|------|
| `/로그인` | 처음 1회 설정 |
| `/시작` | 데이터 수집 + 리포트 생성 |
| `/리포트` | Drive에 저장된 보고서 가져오기 |
| `/자동실행 18:00` | 매일 정해진 시간에 자동 실행 |
| `/자동실행 off` | 자동 실행 해제 |

## 사용 예시

```
1. /로그인
   → 설정 완료

2. /시작
   → 데이터 수집 → 분석 → 리포트 전송

3. /자동실행 18:00
   → 매일 18시에 자동 리포트 생성
```

## 주의사항

- Bot은 항상 실행 중이어야 합니다 (PC 끄면 동작 안 함)
- 처음 1회 `/로그인` 후에는 `/시작`만 입력하면 됩니다
